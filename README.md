# LogX - Behavioral Edge Trading System

LogX is a premium trading journal and behavioral coaching platform designed to help traders identify their psychological edge. Unlike traditional journals, LogX focuses on the intersection of market data and emotional state, providing real-time feedback and coaching interventions.

## ✨ Features

- **Contextual Logging**: Document your emotional state at entry, mid-trade, and exit.
- **Behavioral Analytics**: Correlation between P&L and emotional stability.
- **Smart Coaching**: Identification of tilt patterns and automated interventions.
- **Narrative-Driven UI**: High-end, professional trading environment with zero distraction.
- **Interactive Tour**: Onboarding experience to get you started in seconds.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/project-logx.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production bundle:
```bash
npm run build
```

## 🛠️ Technology Stack

- **Framework**: React 19
- **Build Tool**: Vite 8
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Premium Design System)
- **State Management**: Zustand
- **Icons**: Lucide React
- **Tour**: React Joyride
